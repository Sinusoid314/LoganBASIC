LoopSound(soundName, isLooped)

Description

Makes the sound, specified by soundName, start over when it reaches the end if isLooped is true. Setting isLooped to false will turn looping off.

Usage

print "Loading sound file..." if not loadSound("laff", "../examples/laff.wav") then print "Sound failed to load." end end if print "Sound loaded." print "" loopSound("laff", true) playSound("laff") input("Press enter to quit...") unloadSound("laff")
