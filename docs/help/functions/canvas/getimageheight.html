getImageHeight(imageName)

Description

Returns the height of the image given by imageName.

Usage

var shipWidth, shipHeight loadImage("ship", "../examples/images/ship1.png") shipWidth = getImageWidth("ship") shipHeight = getImageHeight("ship") drawText("Width: " + shipWidth, 10, 10) drawText("Height: " + shipHeight, 10, 25) drawImage("ship", 10, 40) unloadImage("ship") Open in Code Editor
