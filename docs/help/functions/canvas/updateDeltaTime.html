updateDeltaTime()

Description

Sets the current deltaTime to the number of milliseconds since the last call to updateDeltaTime(), and returns that value. If resetDeltaTime() was called prior to calling this function, deltaTime is not changed (remains 0) and the return value is 0.

Usage

var deltaTime var startX = 50, startY = 50 var x = startX, y = startY var width = 100, height = 100 var velocityX = 20, velocityY = 20 hideConsole() enableCanvasBuffer() setFillColor("red") setLineSize(2) mainLoop() wait function reset() x = startX y = startY resetDeltaTime() end function function mainLoop() deltaTime = updateDeltaTime() x = x + (velocityX * deltaTime) y = y + (velocityY * deltaTime) clearCanvas() drawRect(x, y, width, height, true) if x > 100 then reset() drawCanvasBuffer(mainLoop) end function Run | Open in Code Editor
