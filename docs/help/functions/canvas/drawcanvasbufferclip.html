drawCanvasBufferClip(clipX, clipY, clipWidth, clipHeight, drawX, drawY [ ,drawWidth ] [ ,drawHeight ])

Description

Draws part of the offscreen buffer to the visible canvas at the position (drawX, drawY). clipX, clipY, clipWidth, and clipHeight specify the part of the buffer to draw. If drawWidth or drawHeight are specified, the buffer clip is stretched or shrunk to that size.

To set a callback for when the buffer draw has completed, use setCanvasEvent().

Usage

showCanvas() loadImage("ship", "./examples/images/ship1.png") enableCanvasBuffer() input("Press enter to draw image to buffer...") drawImage("ship", 10, 10) input("Press enter to draw buffer clips to canvas...") drawCanvasBufferClip(10, 10, 20, 20, 10, 10) drawCanvasBufferClip(10, 10, 20, 20, 100, 100, 100, 100) input("Press enter to unload image...") unloadImage("ship") print "Done."
