drawRect(drawX, drawY, drawWidth, drawHeight [ ,isFilled ])

Description

Draws a rectangle to the canvas, or the canvas buffer if it is enabled, at the position (drawX, drawY) with the size (drawWidth, drawHeight). If isFilled is true, then the rectangle is filled with the current fill color. If not specified, isFilled defaults to true.

Usage

'Draw a random rectangle var x = int(rnd() * 300) + 1 var y = int(rnd() * 300) + 1 var width = int(rnd() * 100) + 1 var height = int(rnd() * 100) + 1 drawRect(x, y, width, height)
