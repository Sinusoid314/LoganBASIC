getDeltaTime()

Description

Returns the current deltaTime, which is the number of seconds elapsed between the last two calls to updateDeltaTime(), or 0 if resetDeltaTime() was previously called.

Usage

var startX = 50, startY = 50 var x = startX, y = startY var width = 100, height = 100 var velocityX = 20, velocityY = 20 hideConsole() enableCanvasBuffer() setFillColor("red") setLineSize(2) mainLoop() wait function reset() x = startX y = startY resetDeltaTime() end function function mainLoop() updateDeltaTime() x = x + (velocityX * getDeltaTime()) y = y + (velocityY * getDeltaTime()) clearCanvas() drawRect(x, y, width, height, true) if x > 100 then reset() drawCanvasBuffer(mainLoop) end function Run | Open in Code Editor
