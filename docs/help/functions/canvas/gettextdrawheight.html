getTextDrawHeight(text)

Description

Returns the height, in pixels, of text as it would appear when drawn with the current text font.

Usage

var canvasWidth = 400 var canvasHeight = 400 var helloText = "Hello" var worldText = "World!" var drawX = 0 var drawY = 100 setCanvasWidth(canvasWidth) setCanvasHeight(canvasHeight) showCanvas() setTextFont("30px system-ui") drawX = (canvasWidth / 2) - (getTextDrawWidth(helloText) / 2) drawText(helloText, drawX, drawY) setTextFont("48px cursive") drawX = (canvasWidth / 2) - (getTextDrawWidth(worldText) / 2) drawY = drawY + getTextDrawHeight(helloText) drawText(worldText, drawX, drawY) Run | Open in Code Editor
