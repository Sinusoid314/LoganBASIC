drawText(text, drawX, drawY [ ,isFilled ])

Description

Draws text to the canvas, or the canvas buffer if it is enabled, at the position (drawX, drawY). If isFilled is true, then the text is filled with the current fill color. If not specified, isFilled defaults to true.

Usage

var n for n = 10 to 100 step 10 drawText("Hello, world!", n, n) next n Open in Code Editor
