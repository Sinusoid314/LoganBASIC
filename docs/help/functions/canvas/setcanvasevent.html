setCanvasEvent(eventName, [ functionName ])

Description

Sets up a callback function, given by functionName, that will be called whenever a user input event, given by eventName, occurs. If functionName is not specified, then the input event will be removed and no longer trigger a callback. The following is a list of acceptable values for eventName:

  • "pointerdown" - Occurs when a pointer becomes active.
  • "pointerup" - Occurs when a pointer is no longer active.
  • "pointermove" - Occurs when a pointer changes coordinates.
  • "keydown" - Occurs when a keyboard key is pressed down.
  • "keyup" - Occurs when a keyboard key is released.
  • "drawbufferdone" - Occurs after the canvas' offscreen buffer is finished being drawn to the visible canvas via drawCanvasBuffer().

Usage

loadImage("ship", "../examples/images/ship1.png") setCanvasEvent("pointerup", onPointerUp) setCanvasEvent("keyup", onKeyUp) wait function onPointerUp(pointerX, pointerY) drawImage("ship", pointerX, pointerY) end function function onKeyUp(key) if key = "q" then unloadImage("ship") end end if end function Open in Code Editor
