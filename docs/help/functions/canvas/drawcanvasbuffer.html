drawCanvasBuffer()

Description

Transers all graphics from the offscreen buffer to the visible canvas. To set a callback for when the buffer draw has completed, use setCanvasEvent().

Note: If there is already an unfinished buffer draw in progress due to a previous call to drawCanvasBuffer(), the function does nothing.

Usage

showCanvas() loadImage("ship", "../examples/images/ship1.png") enableCanvasBuffer() input("Press enter to draw image to buffer...") drawImage("ship", 10, 10) input("Press enter to draw image to canvas...") disableCanvasBuffer() drawImage("ship", 50, 50) input("Press enter to draw buffer to canvas...") drawCanvasBuffer() unloadImage("ship") print "Done."
