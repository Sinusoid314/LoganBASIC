drawCanvasBuffer([ functionName ])

Description

Transers all graphics from the offscreen buffer to the visible canvas. If functionName is specified, then that function will be called after the buffer has been drawn.

Note: If there is already an unfinished buffer draw in progress due to a previous call to drawCanvasBuffer(), the function does nothing. To set a callback for when a buffer draw has completed without initiating a new draw request, use setCanvasEvent().

Usage

loadImage("ship", "../examples/images/ship1.png") enableCanvasBuffer() input("Press enter to draw image to buffer...") drawImage("ship", 10, 10) input("Press enter to draw image to canvas...") disableCanvasBuffer() drawImage("ship", 50, 50) input("Press enter to draw buffer to canvas...") drawCanvasBuffer() unloadImage("ship") print "Done."
