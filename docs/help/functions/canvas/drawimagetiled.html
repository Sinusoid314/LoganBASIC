drawImageTiled(imageName, drawX, drawY, drawWidth, drawHeight [ ,offsetX ] [ ,offsetY ])

Description

Draws an image specified by imageName to the canvas, or the canvas buffer if it is enabled, in a repeating pattern within the rectangle (drawX, drawY, drawWidth, drawHeight). If offsetX or offsetY are specified, the pattern is shifted by those values.

Usage

var offsetX = 0 var imageWidth, imageHeight hideConsole() loadImage("space", "../examples/images/space.bmp") imageWidth = getImageWidth("space") imageHeight = getImageHeight("space") setCanvasWidth(imageWidth) setCanvasHeight(imageHeight) enableCanvasBuffer() mainLoop() wait function mainLoop() clearCanvas() drawImageTiled("space", 0, 0, imageWidth, imageHeight, -offsetX, 0) if offsetX >= imageWidth then offsetX = offsetX % imageWidth offsetX = offsetX + 5 drawCanvasBuffer(mainLoop) end function Open in Code Editor
