drawCircle(centerX, centerY, radius [ ,isFilled ])

Description

Draws a circle to the canvas, or the canvas buffer if it is enabled, with center at (centerX, centerY) and radius given by radius. If isFilled is true, then the circle is filled with the current fill color. If not specified, isFilled defaults to true.

Usage

'Draw a random circle var centerX = int(rnd() * 300) + 1 var centerY = int(rnd() * 300) + 1 var radius = int(rnd() * 100) + 1 drawCircle(centerX, centerY, radius) Open in Code Editor
