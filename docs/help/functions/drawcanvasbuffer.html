drawCanvasBuffer([ functionName ])

Description

Transers all graphics from the offscreen buffer to the visible canvas. If functionName is specified, then that function will be called after the buffer has been drawn.

Usage

loadImage("fish", "../examples/fishy-fart-os.png") enableCanvasBuffer() input("Press enter to draw image to buffer...") drawImage("fish", 10, 10) input("Press enter to draw image to canvas...") disableCanvasBuffer() drawImage("fish", 50, 50) input("Press enter to draw buffer to canvas...") drawCanvasBuffer() unloadImage("fish") print "Done."
