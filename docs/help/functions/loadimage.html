loadImage(imageName, imageSource)

Description

Loads an image file specified by imageSource and assigns it a reference name specified by imageName. The function waits for the image to load before returning, and returns true if the image successfully loaded, and false otherwise. Once loaded, the image can be drawn on the canvas with DrawImage and unloaded with UnloadImage.

Usage

loadImage("ship", "../examples/ship1.png") input("Press enter to draw image...") drawImage("ship", 10, 10) input("Press enter to unload image...") unloadImage("ship") print "Done."
