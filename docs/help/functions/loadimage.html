LoadImage(imageName, imageSource)

Description

Loads an image file specified by imageSource and assigns it a reference name specified by imageName. Once loaded, the image can be drawn on the canvas with DrawImage and unloaded with UnloadImage.

Usage

loadImage("fish", "../examples/fishy-fart-os.png") input("Press enter to draw image...") drawImage("fish", 10, 10) input("Press enter to unload image...") unloadImage("fish") print "Done."
