unloadSound(soundName)

Description

Unloads a sound, specified by soundName, that was loaded using LoadSound.

Usage

var choice print "Loading sound file..." if not loadSound("mac_intro", "../examples/mac_intro.wav") then print "Sound failed to load." end end if print "Sound loaded." do print "" print "Enter 'play', 'pause', 'stop', or 'q' to quit." choice = input(": ") if choice = "play" then playSound("mac_intro") if choice = "pause" then pauseSound("mac_intro") if choice = "stop" then stopSound("mac_intro") loop while choice <> "q" unloadSound("mac_intro") print "" print "Done."
