drawRect(drawX, drawY, drawWidth, drawHeight)

Description

Draws a rectangle at the position (drawX, drawY) with the size (drawWidth, drawHeight), with no fill color.

Usage

'Draw a random rectangle var x = int(rnd() * 300) + 1 var y = int(rnd() * 300) + 1 var width = int(rnd() * 100) + 1 var height = int(rnd() * 100) + 1 drawRect(x, y, width, height)
