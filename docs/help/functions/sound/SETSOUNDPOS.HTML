setSoundPos(soundName, position)

Description

Sets the position to start play from in the sound specified by soundName. The position can be any number between 0 and the length of the sound. To get the sound's length, use GetSoundLen.

Usage

var position print "Loading sound file..." if not loadSound("mac", "../examples/sounds/mac.wav") then print "Sound failed to load." end end if print "Sound loaded." print "" playSound("mac") position = input("Enter a position to jump to: ") setSoundPos("mac", position) input("Press enter to quit...") unloadSound("mac") Open in Code Editor
