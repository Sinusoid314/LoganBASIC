getSoundPos(soundName)

Description

Returns the current position of the sound specified by soundName.

Usage

var position print "Loading sound file..." if not loadSound("mac", "../examples/sounds/mac.wav") then print "Sound failed to load." end end if print "Sound loaded." print "" playSound("mac") input("Press enter to get the sound's current position...") position = getSoundPos("mac") print "Sound position: " + position unloadSound("mac")
