loadSound(soundName, soundSource)

Description

Loads an audio file specified by soundSource and assigns it a reference name specified by soundName. The function waits for the sound to load before returning, and returns true if the sound successfully loaded, and false otherwise. Once loaded, the sound can be played with PlaySound and unloaded with UnloadSound.

Usage

var choice print "Loading sound file..." if not loadSound("mac", "../examples/mac.wav") then print "Sound failed to load." end end if print "Sound loaded." do print "" print "Enter 'play', 'pause', 'stop', or 'q' to quit." choice = input(": ") if choice = "play" then playSound("mac") if choice = "pause" then pauseSound("mac") if choice = "stop" then stopSound("mac") loop while choice <> "q" unloadSound("mac") print "" print "Done."
