getImageWidth(imageName)

Description

Returns the width of the image given by imageName.

Usage

var fishWidth, fishHeight loadImage("fish", "../examples/fishy-fart-os.png") fishWidth = getImageWidth("fish") fishHeight = getImageHeight("fish") drawText("Width: " + fishWidth, 10, 10) drawText("Height: " + fishHeight, 10, 25) drawImage("fish", 10, 40) unloadImage("fish")
