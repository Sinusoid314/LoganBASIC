import(sourceFile)

Description

Downloads, compiles, and runs the source code contained in sourceFile. The function waits for the code to finish running before returning. If successful, it returns true. If sourceFile fails to download, it returns false. If the code fails to compile or run, and error is raised.

Usage

importExample.bas print "This is the imported file." example.bas print "Hello!" import("importExample.bas") print "Goodbye!" 'output: 'Hello! 'This is the imported file. 'Goodbye! Run | Open in Code Editor
