rectOverlapsCircle(rectX, rectY, rectWidth, rectHeight, circleX, circleY, circleRadius)

Description

Returns true if the rectangle (rectX, rectY, rectWidth, rectHeight) and the circle (circleX, circleY, circleRadius) overlap, and false otherwise.
