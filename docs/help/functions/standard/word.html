word(string, position, [ delimiter ])

Description

Returns the word at the given position within string. If position is less than 1 or greater than the number of words in string, then an empty string is returned. If delimiter is ommited, then the space character is used as the word separator.

Usage

print word("first second third", 2) print word("red,green,blue", 1, ",") 'output: 'second 'red
