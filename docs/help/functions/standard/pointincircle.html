pointInCircle(pointX, pointY, circleX, circleY, circleRadius)

Description

Returns true if the point (pointX, pointY) is within the circle (circleX, circleY, circleRadius), and false otherwise.

Usage

var circleX = 100 var circleY = 100 var circleRadius = 50 hideConsole() setFillColor("red") setLineSize(2) drawCircle(circleX, circleY, circleRadius, false) setCanvasEvent("pointerup", onPointerUp) wait function onPointerUp(pointerX, pointerY) clearCanvas() if pointInCircle(pointerX, pointerY, circleX, circleY, circleRadius) then drawCircle(circleX, circleY, circleRadius, true) else drawCircle(circleX, circleY, circleRadius, false) end if end function Open in Code Editor
