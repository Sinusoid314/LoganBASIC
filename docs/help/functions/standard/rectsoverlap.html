rectsOverlap(rectX1, rectY1, rectWidth1, rectHeight1, rectX2, rectY2, rectWidth2, rectHeight2)

Description

Returns true if the rectangles (rectX1, rectY1, rectWidth1, rectHeight1) and (rectX2, rectY2, rectWidth2, rectHeight2) overlap, and false otherwise.
