run(source)

Description

Compiles and runs the source code contained in source. If the source fails to either compile or run properly, an error is raised; otherwise the function waits for the code to finish running before returning. source is compiled and run in the global context, so it does not have access to any local variables, even if the run() function is called within a function.

Usage

var greeting = "Hello, world!" run("print greeting") 'output: 'Hello, world! Run | Open in Code Editor
