circlesOverlap(circleX1, circleY1, circleRadius1, circleX2, circleY2, circleRadius2)

Description

Returns true if the circles (circleX1, circleY1, circleRadius1) and (circleX2, circleY2, circleRadius2) overlap, and false otherwise.
