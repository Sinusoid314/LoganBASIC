timerIsRunning()

Description

Returns true if the timer is currently running, and false otherwise. The timer starts running after calling startTimer(), and stops running after calling stopTimer().

Usage

var tickCount = 0 var choice = "" do print "" print "Enter 'start', 'stop', or 'q' to quit." choice = input(": ") if choice = "start" then if not timerIsRunning() then startTimer(1000, onTick) end if if choice = "stop" then stopTimer() loop while choice <> "q" stopTimer() print "" print "Done." function onTick() tickCount = tickCount + 1 print "Tick " + tickCount + " at " + time() end function Run | Open in Code Editor
