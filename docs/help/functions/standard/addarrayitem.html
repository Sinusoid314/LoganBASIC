addArrayItem(arrayName, itemVal, [ beforeIndex ])

Description

Adds itemVal to the array given by arrayName before the array item given by beforeIndex. If beforeIndex is not specified, itemVal is added to the end of the array. This function only works on one-dimensional arrays.

Usage

array myArray[2] var index myArray[0] = "Apple" myArray[1] = "Orange" for index = 0 to len(myArray) - 1 print myArray[index] next index print "" addArrayItem(myArray, "Pear", 1) for index = 0 to len(myArray) - 1 print myArray[index] next index 'output: 'Apple 'Orange ' 'Apple 'Pear 'Orange
