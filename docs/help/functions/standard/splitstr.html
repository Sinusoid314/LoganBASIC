splitStr(string, delimiter)

Description

Splits string into a group of sub-strings, seperated by delimiter, and returns them in an array.

Usage

var text = "one two three four" var words = splitStr(text, " ") var n for n = 0 to len(words) - 1 print words[n] next n 'output: 'one 'two 'three 'four Open in Code Editor
