DrawImage(imageName, drawLeft, drawTop)

Description

Draws an image specified by imageName to the canvas, or the canvas buffer if it is enabled, at the position (drawLeft, drawTop).

Usage

loadImage("fish", "../examples/fishy-fart-os.png") input("Press enter to draw image...") drawImage("fish", 10, 10) input("Press enter to unload image...") unloadImage("fish") print "Done."
