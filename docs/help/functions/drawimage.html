drawImage(imageName, drawX, drawY [ ,drawWidth ] [ ,drawHeight ])

Description

Draws an image specified by imageName to the canvas, or the canvas buffer if it is enabled, at the position (drawX, drawY). If drawWidth or drawHeight are specified, the image is stretched or shrunk to that size.

Usage

loadImage("ship", "../examples/ship1.png") input("Press enter to draw image...") drawImage("ship", 10, 10) drawImage("ship", 100, 100, 100, 100) input("Press enter to unload image...") unloadImage("ship") print "Done."
