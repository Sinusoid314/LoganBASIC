drawImageClip(imageName, clipX, clipY, clipWidth, clipHeight, drawX, drawY [ ,drawWidth ] [ ,drawHeight ])

Description

Draws part of an image specified by imageName to the canvas, or the canvas buffer if it is enabled, at the position (drawX, drawY). clipX, clipY, clipWidth, and clipHeight specify the part of the image to draw. If drawWidth or drawHeight are specified, the image is stretched or shrunk to that size.

Usage

loadImage("fish", "../examples/fishy-fart-os.png") input("Press enter to draw image...") drawImageClip("fish", 10, 10, 20, 20, 10, 10) drawImageClip("fish", 10, 10, 20, 20, 100, 100, 100, 100) input("Press enter to unload image...") unloadImage("fish") print "Done."
