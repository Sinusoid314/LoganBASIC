loadSpriteSheet(sheetName, sheetSource, columnCount, rowCount)

Description

Loads an image file specified by sheetSource and assigns it a reference name specified by sheetName. The function waits for the image to load before returning, and returns true if the image successfully loaded, and false otherwise. Once loaded, the sheet image can be used by a sprite created with addSprite, and unloaded with unloadSpriteSheet.
