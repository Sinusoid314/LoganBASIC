loadSpriteSheet(sheetName, sheetSource, columnCount, rowCount)

Description

Loads an sprite sheet image file specified by sheetSource and assigns it a reference name specified by sheetName. The function waits for the image to load before returning, and returns true if the image successfully loaded, and false otherwise. Once loaded, the sheet image can be used by a sprite created with addSprite, and unloaded with unloadSpriteSheet.

Usage

var deltaTime, prevTime hideConsole() enableCanvasBuffer() loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) addSprite("bubba", "bubba-sheet", 10, 100) setSpriteFrameRange("bubba", 0, 3) setSpriteVelocityX("bubba", 60) setSpriteFrameRate("bubba", 8) prevTime = time() mainLoop() wait function mainLoop() deltaTime = (time() - prevTime) / 1000 prevTime = time() if getSpriteX("bubba") >= 300 then removeSprite("bubba") unloadSpriteSheet("bubba-sheet") end end if updateSprites(deltaTime) clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function Open in Code Editor
