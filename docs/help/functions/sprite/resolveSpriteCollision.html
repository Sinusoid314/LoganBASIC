resolveSpriteCollision(spriteName1, spriteName2, contactTime)

Description

Moves spriteName1 and spriteName2 to the positions they had at the point in time, given by contactTime, when both sprites first made contact after the last call to updateSprites(). contactTime is a value greater than or equal to 0, and less than 1, that is obtained by calling spritesCollided().

Usage

Run | Open in Code Editor
