getSpriteCycles(spriteName)

Description

Returns the number of times spriteName cycles through its animation before stopping. A value of zero indicates that the animation never stops.

Usage

var isDone = false hideConsole() enableCanvasBuffer() loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) addSprite("bubba", "bubba-sheet", 50, 50) setSpriteFrameRange("bubba", 0, 3) setSpriteFrameRate("bubba", 6) setCanvasEvent("pointerup", onPointerUp) mainLoop() wait function mainLoop() if isDone then removeSprite("bubba") unloadSpriteSheet("bubba-sheet") end end if updateSprites() clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function function onPointerUp(pointerX, pointerY) if pointInSprite("bubba", pointerX, pointerY) then setSpriteCycles("bubba", (getSpriteCycles("bubba") + 1) % 6) setSpritePlaying("bubba", true) else isDone = true end if end function Open in Code Editor
