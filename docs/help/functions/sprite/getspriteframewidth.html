getSpriteFrameWidth(spriteName)

Description

Returns the width of spriteName's animation frames.

Usage

var scale = 0 hideConsole() enableCanvasBuffer() loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) addSprite("bubba", "bubba-sheet", 10, 10) setSpriteFrameRange("bubba", 0, 0) mainLoop() wait function mainLoop() if getSpriteDrawWidth("bubba") >= 150 then removeSprite("bubba") unloadSpriteSheet("bubba-sheet") end end if scale = scale + 0.1 setSpriteDrawWidth("bubba", getSpriteFrameWidth("bubba") * scale) clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function
