getSpriteScroll(spriteName)

Description

Returns true if the scroll offset is applied to spriteName's position when drawing the sprites, and false otherwise. This property is set to true by default.

Usage

hideConsole() enableCanvasBuffer() loadSpriteSheet("ship-sheet", "../examples/images/ship-sheet.png", 2, 1) addSprite("ship", "ship-sheet", 0, 0) setScrollX(-100) setScrollY(-100) setCanvasEvent("pointerup", onPointerUp) setCanvasEvent("keyup", onKeyUp) mainLoop() wait function mainLoop() clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function function onPointerUp(pointerX, pointerY) setSpriteScroll("ship", not getSpriteScroll("ship")) end function function onKeyUp(key) removeSprite("ship") unloadSpriteSheet("ship-sheet") end end function
