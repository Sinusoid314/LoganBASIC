setSpriteDrawWidth(spriteName, drawWidth)

Description

Sets the display width of spriteName to drawWidth.

Usage

hideConsole() enableCanvasBuffer() loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) addSprite("bubba", "bubba-sheet", 10, 10) setSpriteFrameRange("bubba", 0, 0) mainLoop() wait function mainLoop() if getSpriteDrawWidth("bubba") >= 150 then removeSprite("bubba") unloadSpriteSheet("bubba-sheet") end end if setSpriteDrawWidth("bubba", getSpriteDrawWidth("bubba") + 1) clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function
