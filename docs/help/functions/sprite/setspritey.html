setSpriteY(spriteName, y)

Description

Sets the vertical position of spriteName to y.

Usage

var deltaTime, prevTime hideConsole() enableCanvasBuffer() loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) addSprite("bubba", "bubba-sheet", 0, 10) setSpriteFrameRange("bubba", 4, 4) setSpriteVelocityY("bubba", 60) prevTime = time() mainLoop() wait function mainLoop() deltaTime = (time() - prevTime) / 1000 prevTime = time() if getSpriteY("bubba") = 100 then removeSprite("bubba") unloadSpriteSheet("bubba-sheet") end end if updateSprites(deltaTime) if getSpriteY("bubba") > 50 then setSpriteY("bubba", 100) end if clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function
