setSpriteDrawHeight(spriteName, drawHeight)

Description

Sets the display height of spriteName to drawHeight.

Usage

hideConsole() enableCanvasBuffer() loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) addSprite("bubba", "bubba-sheet", 10, 10) setSpriteFrameRange("bubba", 0, 0) mainLoop() wait function mainLoop() if getSpriteDrawHeight("bubba") >= 270 then removeSprite("bubba") unloadSpriteSheet("bubba-sheet") end end if setSpriteDrawHeight("bubba", getSpriteDrawHeight("bubba") + 1) clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function Open in Code Editor
