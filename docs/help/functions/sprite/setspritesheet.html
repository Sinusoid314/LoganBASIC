setSpriteSheet(spriteName, sheetName)

Description

Sets the sprite sheet used by spriteName to sheetName. The sprite's current animation frame index is reset to zero, its draw size is set to the size of the sheet's frames, and its frame range is reset to encompass all of the sheet's frames.

Usage

var deltaTime, prevTime hideConsole() enableCanvasBuffer() loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) loadSpriteSheet("ship-sheet", "../examples/images/ship-sheet.png", 2, 1) addSprite("sprite", "bubba-sheet", 10, 100) setSpriteFrameRange("sprite", 0, 3) setSpriteVelocityX("sprite", 60) setSpriteFrameRate("sprite", 8) prevTime = time() mainLoop() wait function mainLoop() deltaTime = (time() - prevTime) / 1000 prevTime = time() if getSpriteX("sprite") >= 150 then setSpriteSheet("sprite", "ship-sheet") end if if getSpriteX("sprite") >= 300 then removeSprite("sprite") unloadSpriteSheet("bubba-sheet") unloadSpriteSheet("ship-sheet") end end if updateSprites(deltaTime) clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function
