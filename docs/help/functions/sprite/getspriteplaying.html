getSpritePlaying(spriteName)

Description

Returns true if spriteName's animation is playing, and false if it is stopped.

Usage

var isDone = false hideConsole() enableCanvasBuffer() loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) addSprite("bubba", "bubba-sheet", 50, 50) setSpriteFrameRange("bubba", 0, 3) setSpriteFrameRate("bubba", 8) setCanvasEvent("pointerup", onPointerUp) mainLoop() wait function mainLoop() if isDone then removeSprite("bubba") unloadSpriteSheet("bubba-sheet") end end if updateSprites() clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function function onPointerUp(pointerX, pointerY) if pointInSprite("bubba", pointerX, pointerY) then setSpritePlaying("bubba", not getSpritePlaying("bubba")) else isDone = true end if end function Run | Open in Code Editor
