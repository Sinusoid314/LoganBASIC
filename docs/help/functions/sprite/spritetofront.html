spriteToFront(spriteName)

Description

Brings spriteName to the top of the z-order, so it is displayed over all other sprites.

Usage

hideConsole() enableCanvasBuffer() loadSpriteSheet("ship-sheet", "../examples/images/ship-sheet.png", 2, 1) loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) addSprite("ship", "ship-sheet", 30, 60) addSprite("bubba", "bubba-sheet", 30, 30) setCanvasEvent("pointerup", onPointerUp) setCanvasEvent("keyup", onKeyUp) mainLoop() wait function mainLoop() clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function function onPointerUp(pointerX, pointerY) spriteToFront("ship") end function function onKeyUp(key) removeSprite("ship") removeSprite("bubba") unloadSpriteSheet("ship-sheet") unloadSpriteSheet("bubba-sheet") end end function
