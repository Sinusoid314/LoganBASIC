setSpriteFrameRate(spriteName, frameRate)

Description

Sets the animation rate of spriteName to frameRate. The higher the rate number is, the slower the sprite's animation will be.

Usage

var isDone = false hideConsole() enableCanvasBuffer() loadSpriteSheet("bubba-sheet", "../examples/images/bubba.png", 5, 2) addSprite("bubba", "bubba-sheet", 50, 50) setSpriteFrameRange("bubba", 0, 3) setSpriteFrameRate("bubba", 1) setCanvasEvent("pointerup", onPointerUp) mainLoop() wait function mainLoop() if isDone then removeSprite("bubba") unloadSpriteSheet("bubba-sheet") end end if updateSprites() clearCanvas() drawSprites() drawCanvasBuffer(mainLoop) end function function onPointerUp(pointerX, pointerY) if pointInSprite("bubba", pointerX, pointerY) then setSpriteFrameRate("bubba", (getSpriteFrameRate("bubba") + 1) % 10) else isDone = true end if end function Open in Code Editor
