Debugger Commands

The debugger commands let you tell the debugger when and how to step through your program. They are located at the top of the debugger pane:

Resume

Runs the program at regular speed without pausing, unless a breakpoint is encountered.

Pause

Pauses the program execution, highlights the next line of code to be run, and updates the variable and call stack displays.

Step Into

Runs the highlighted line of code, updates the variable and call stack displays, and highlights the next line of code to be run. If the highlighted line contains any function calls, the debugger will pause at the beginning of each function call.

Step Over

Same as Step Into, except the debugger will run through every function call in the line without pausing.

Step Out

Runs all the code from the currently highlighted line to the end of the current function, pausing only after the function returns.

Skip

Skips over the currently highlighted line without running any of its code.
