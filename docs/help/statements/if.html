If expression Then statement If expression Then statements [ Else statements ] End If

Description

Conditionally executes a block of statements, depending on the value of expression.

Usage

var num = 0 num = input("Enter a number: ") if num = 10 then print "Number is 10." else if num < 10 then print "Number is less than 10." else print "Number is greater than 10." end if end if
