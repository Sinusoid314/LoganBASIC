Function functionName( [ parameter1, parameter2, ... ] ) statements End Function

Description

Defines a block of statements that can be called from anywhere in the program. Functions can optionally take any number of argument values from, and return a value to, the function's caller.

Usage

print getGreeting("Alice") print getGreeting("Bob") function getGreeting(name) var greeting = "Hello, " + name + "!" return greeting end function 'output: 'Hello, Alice! 'Hello, Bob! Open in Code Editor
