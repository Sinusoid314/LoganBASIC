Structure structName field1 field2 ... End Structure

Description

Defines a set of related variables, called fields, that are accessed and modified through a single variable. New instances of a structure are created with the New operator.

Usage

structure Rectangle x y width height end structure array rects[10] var n 'Create 10 random rectangles for n = 0 to 9 rects[n] = new Rectangle rects[n].x = int(rnd() * 300) + 1 rects[n].y = int(rnd() * 300) + 1 rects[n].width = int(rnd() * 100) + 1 rects[n].height = int(rnd() * 100) + 1 next n 'Draw the rectangles for n = 0 to 9 drawRect(rects[n].x, rects[n].y, rects[n].width, rects[n].height) next n Open in Code Editor
