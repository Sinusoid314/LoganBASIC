New [ Array[expression1, expression2, ...] | identifier ]

Description

Creates either a new instance of a Structrue, or a new anonymous Array.

Usage

structure Employee name id end structure 'Create a new anonymous array and assign it to a variable var employees = new array[3] 'Create 3 new employees var n for n = 0 to 2 employees[n] = new Employee employees[n].id = n + 1 next n employees[0].name = "Alice" employees[1].name = "Bob" employees[2].name = "Chris" for n = 0 to 2 print "Employee #" + employees[n].id + " - " + employees[n].name print "" next n 'output: 'Employee #1 - Alice 'Employee #2 - Bob 'Employee #3 - Chris
