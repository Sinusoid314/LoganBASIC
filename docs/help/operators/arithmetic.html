Arithmetic Operators

Operator Description Usage
+ Adds two numbers. print 2 + 2
'output: 4
- Subtracts the second number from the first. print 10 - 2
'output: 8
* Multiplies two numbers. print 3 * 4
'output: 12
/ Divides the first number by the second. print 12 / 3
'output: 4
% Returns the remainder of dividing the first number by the second. print 5 % 2
'output: 1
^ Raises the first number to the power of the second. print 5^2
'output: 25
- (unary) Switches the sign of a number. print -4
'output: -4
