Comparison Operators

Operator Description Usage
= Returns True if both values, string or number, are equal, and False otherwise. print (2 + 2) = 5
'output: false
<> Returns True if both values, string or number, are not equal, and False otherwise. print 4 <> 5
'output: true
> Returns True if the first number is greater than the second, and False otherwise. print 10 > 5
'output: true
>= Returns True if the first number is greater than or equal to the second, and False otherwise. print 20 >= 10
'output: true
< Returns True if the first number is less than the second, and False otherwise. print 10 < 5
'output: false
<= Returns True if the first number is less than or equal to the second, and False otherwise. print 20 <= 10
'output: false
